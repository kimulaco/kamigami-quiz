# 神々のクイズ

「[神の一覧 - Wikipedia](https://ja.wikipedia.org/wiki/%E7%A5%9E%E3%81%AE%E4%B8%80%E8%A6%A7)」の情報を元に作成されたジョーククイズサイト。

[https://kmrk.github.io/kamigami-quiz/](https://kmrk.github.io/kamigami-quiz/)

## Development

``` bash
# install
$ yarn install

# Launch develop Server
$ yarn run dev

# Build SPA Files
$ yarn run build
$ yarn start

# Update quiz Data
$ yarn run update
```
